# 03 ターゲットユーザー

## メインターゲット
- 20代前半の大学生・専門学生
- 就活を本格的に始めた人
- 複数企業を並行で受ける人

## ユーザー像

### ユーザーA
- 複数社のES締切を同時に管理している
- 面接日程や企業研究メモがバラバラになりがち
- 情報の一元化をしたい

### ユーザーB
- 面接で毎回似た質問をされるが、過去の記録を活かせていない
- 改善を積み上げて選考通過率を上げたい

### ユーザーC
- Notionのような整理方法は魅力だが、自分で初期構成を作るのが面倒
- 最初から就活向けに整ったサービスを使いたい

## 利用シーン
- 企業研究をまとめる
- ES締切とタスクを管理する
- 面接前に企業別メモを見返す
- 面接後に振り返りを記録する
- 選考進捗を一覧で確認する
