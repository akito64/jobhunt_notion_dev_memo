# 11 MVP定義

## MVPの考え方
MVPでは「Notionを完全再現すること」ではなく、「就活管理が実際に成立すること」を重視する。

## 必須機能
- ログイン / 新規登録
- ダッシュボード
- 企業一覧 / 詳細
- 選考ステータス管理
- ES保存
- 面接ログ保存
- カレンダー表示

## あれば強い機能
- テンプレートから新規作成
- チェックリスト付きページ
- タスク管理

## 後回しにする機能
- リアルタイム共同編集
- 高度な権限管理
- 複雑なブロック種類
- 高度な通知機能
- 大規模AI自動化

## 完成判定
- 実際に1人の学生が複数企業を管理できる
- 予定と選考進捗が見える
- ESと面接ログを保存できる
- スマホで確認できる
