# 04 機能一覧

## コア機能

### 1. 認証機能
- 新規登録
- ログイン
- ログアウト
- パスワード再設定

### 2. ダッシュボード
- 今週の締切表示
- 直近の面接予定表示
- エントリー中企業数
- 未完了タスク数
- 選考状況サマリー

### 3. 企業管理
- 企業の新規作成
- 企業情報の編集
- 業界、志望度、選考状況の登録
- 企業ごとのメモ保存

### 4. 選考管理
- ステータス管理
- カンバン表示
- 一覧表示
- ステータス変更履歴

### 5. ES管理
- 自己PR、ガクチカ、志望動機の保存
- 企業別ES紐付け
- 下書き保存
- 提出版の履歴管理

### 6. 面接ログ
- 面接日
- 面接形式
- 聞かれた質問
- 自分の回答
- 改善点
- 次回の準備事項

### 7. カレンダー
- 面接予定
- 説明会
- 締切
- タスク期限

### 8. ページ機能
- ページ作成
- タイトル編集
- 見出し / 箇条書き / チェックリスト
- テンプレートから作成

## 拡張候補機能
- AIによるES添削
- 面接想定質問生成
- 共有リンク
- コメント機能
- 通知機能
- タグ機能
