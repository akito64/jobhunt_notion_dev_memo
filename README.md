# Shukatsu OS

就活に必要な情報・タスク・企業研究・ES・面接記録を、1つのワークスペースで管理できる **就活特化のNotion風Webアプリ** です。

## このリポジトリの目的
このリポジトリは、卒業制作における企画・設計・実装・公開・運用までをチームで進めるための共通ベースです。

## コンセプト
- Notionのようにページを作って情報を整理できる
- ただし汎用メモではなく、就活に必要な機能に特化する
- 学生が実際に使える完成度を目指す

## 解決したい課題
就活では以下の情報が複数のサービスやメモに分散しやすいです。

- 企業研究
- ESの下書きと提出管理
- 面接日程
- 面接で聞かれたこと
- 次回までの改善点
- タスクや締切

その結果、管理コストが高くなり、抜け漏れや準備不足が起きやすくなります。

## 目指す価値
Shukatsu OSでは、就活に必要な情報を1つの画面群でつなぎ、

- 見える化
- 一元管理
- 行動につながる整理
- 改善の積み上げ

を実現します。

## 想定ユーザー
- 就活を始めた大学生・専門学生
- 複数企業を並行管理している学生
- ESや面接の記録を体系的に残したい人
- 就活管理をスマホでも見たい人

## MVPで作るもの
最初の完成目標は以下です。

- ログイン機能
- ダッシュボード
- 企業管理ページ
- 選考カンバン
- ES管理
- 面接ログ
- カレンダー
- テンプレート機能

## 技術候補
- Frontend: Next.js
- UI: Tailwind CSS / shadcn/ui
- Backend / DB: Supabase
- Hosting: Vercel
- Editor: Tiptap または Editor.js
